# Bispil

Lah mich dir es paar Bispil zeige:

## Grundlegendi Dokumentstruktur

Eins vo de erste Sache, wo mich beeindruckt het, isch gsi, wie sauber und intuitiv d Syntax isch. Do isch es eifachs Bispil:

```typst
#set page(margin: 2cm)
#set text(font: "Linux Libertine", size: 11pt)

= Iileitig
Das isch mini Dokumentation über das, was ich i dere Abteilig glernt ha.

== Erste Abschnitt
Do cha ich über mini Erfahrige und Erkenntnis schriibe.
```

D `#set` Direktive ermöglicht dir, s Dokument global z konfiguriere, und d Überschrifte-Syntax isch so viel sauberer als LaTeX's `\section{}` Befähl.

## Benutzerdefinierti Vorlage

S Erstelle vo benutzerdefinierte Vorlage isch es Chinderspiel. Du chasch wiederverwendbari Komponente definiere:

```typst
#let task(title, description) = {
  block(
    width: 100%,
    fill: rgb("f0f0f0"),
    radius: 8pt,
    padding: 12pt,
    [
      *Ufgab: #title*
      #description
    ]
  )
}

#task(
  title: "Typst lerne",
  description: "Word-Dokument in Typst-Format konvertiere"
)
```

De funktional Ansatz macht es eifach, konsistenti, wiederverwendbari Dokumentelement z erstelle.

## Tabälle und Liste

Tabälle sind unkompliziert und lesbar:

```typst
#table(
  columns: 3,
  [*Datum*], [*Ufgab*], [*Status*],
  [2025-08-15], [Typst iirichte], [Erledigt],
  [2025-08-16], [Dokumentation schriibe], [I Bearbeitig],
)
```

Und Liste funktioniere genau wie du es in Markdown erwarte würdsch, aber mit meh Kontrolle:

```typst
- Erste Punkt
- Zweite Punkt
  - Verschachtelte Punkt
  - No en verschachtelte Punkt
- Dritte Punkt
```
